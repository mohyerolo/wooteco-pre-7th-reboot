### 첫 번째 미션: 계산기

[1주 차 미션 진행 깃허브](https://github.com/mohyerolo/java-calculator-7/tree/mohyerolo)

#### 목표

1. MVC 패턴 적용
2. 클래스, 메서드 역할 분리
3. 객체 지향

#### 구현 기능

- 문자열 분리
    - 쉼표(,) 또는 콜론(:) 을 기준으로 분리
    - 커스텀 구분자 지정 가능: "//"와 "\n" 사이에 위치시키는 문자가 커스텀 구분자가 됨
- 분리된 문자를 숫자로 변환 후 각 숫자의 합을 반환
- 잘못된 값이 들어오면 IllegalArgumentException을 발생시키고 애플리케이션 종료

#### 구현 순서

1. 문자 받기
2. 문자 형태 검사
3. 문자 분리
4. 분리된 문자 검사
5. 분리된 문자 숫자 변환, 합 계산
